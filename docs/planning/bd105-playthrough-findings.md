# BD-105: Automated playthrough to 6-minute mark — findings document

**Category:** QA / Playtesting
**Priority:** P1
**File(s):** Output: `play-through-findings.md`

## Description
Run one automated playthrough of the 3D Roguelike Survivor mode through the 6-minute mark. Capture screenshots at key moments. Document findings for improving the game and identifying bugs in `play-through-findings.md`.

## Scope
- Start a new game (any animal, any difficulty)
- Play through at least 6 minutes of gameplay
- Capture screenshots at: game start, first wave, first loot drop, first upgrade menu, minimap view, and late-game combat
- Document: visual bugs, balance issues, UX problems, gameplay feel, suggestions for improvement

## Deliverable
`play-through-findings.md` in project root
